# GoodToEat - Handcrafted Irish Ghee Website

An interactive website for GoodToEat company showcasing premium handcrafted Irish ghee.

## Features

- **Home Page**: Welcome page with company overview and key features
- **Products Page**: Detailed product information with ordering functionality
- **About Page**: Company story, mission, values, and process
- **Contact Us Page**: Contact information and interactive contact form

## Installation

1. Install the required packages:
```bash
pip install -r requirements.txt
```

## Running the Application

To run the Streamlit app locally:

```bash
streamlit run app.py
```

The app will open in your default web browser at `http://localhost:8501`

## Deployment

### Deploy to Streamlit Cloud

1. Push your code to a GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with your GitHub account
4. Click "New app"
5. Select your repository and branch
6. Set the main file path to `app.py`
7. Click "Deploy"

### Alternative Deployment Options

- **Heroku**: Use the Procfile and requirements.txt
- **AWS/Azure/GCP**: Deploy as a containerized application
- **Docker**: Create a Dockerfile for container deployment
